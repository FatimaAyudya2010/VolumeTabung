# VolumeTabung #

# Deskripsi Program
Program VolumeTabung digunakan untuk menghitung volume tabung berdasarkan nilai jari-jari alas dan tinggi tabung yang dimasukkan oleh pengguna.
Program ini menerapkan konsep fungsi (method) dalam Java dengan menggunakan parameter untuk menghitung volume menggunakan rumus matematika tabung.

# Cara Kerja Program
> Input Data dari Pengguna
- Program meminta dua nilai melalui keyboard:
- Jari-jari tabung (r) dalam satuan sentimeter
- Tinggi tabung (t) dalam satuan sentimeter
> Pemanggilan Fungsi hitungVolume()
- Program memanggil fungsi:
hitungVolume(r, t)
- Fungsi tersebut menghitung volume menggunakan rumus:
Volume = π × r² × t
- Nilai π diambil dari konstanta Math.PI.
> Output Hasil
- Program menampilkan hasil perhitungan volume dalam satuan cm³ (sentimeter kubik), ditampilkan dengan dua angka di belakang koma menggunakan printf.

# Contoh Input dan Output
> Input:
Masukkan jari-jari tabung (cm): 7
Masukkan tinggi tabung (cm): 10
> Output:
Volume tabung adalah: 1539.38 cm³

# Konsep yang Digunakan
> Fungsi (Method) dengan Parameter dan Return Value
Fungsi hitungVolume(double jariJari, double tinggi) digunakan untuk melakukan perhitungan dan mengembalikan hasil.
> Kelas Scanner untuk membaca input dari pengguna.
> Konstanta Math.PI untuk nilai π (pi).
> Format output menggunakan System.out.printf agar hasil lebih rapi.
